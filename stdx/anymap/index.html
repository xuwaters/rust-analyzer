This implementation provides a safe and convenient store for one value of each type.

Your starting point is Map. It has an example.

§Cargo features

This implementation has two independent features, each of which provides an implementation providing types Map, AnyMap, OccupiedEntry, VacantEntry, Entry and RawMap:

  • std (default, enabled in this build): an implementation using std::collections::hash_map, placed in the crate root (e.g. anymap::AnyMap).
